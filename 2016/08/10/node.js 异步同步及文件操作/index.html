node.js 异步同步及文件操作

一 ES6
ES6也叫2015 是javascript最新的版本，有着丰富的新特性。
ES5是目前所有浏览器都支持的javascript版本
ES6只有chrome等一些浏览器的最新版才支持 但支持的很快 因为有babael这样的javascript的贬义工具 ES6一快速大量的用于生产中。

字符串模板
ES5里字符串的拼接很烦
ES6推出字符串模板功能
${变量}
__dirname node.js当前文件目录地址

const 常量声明
let 块域作用域变量，替代var

require node.js 引入文件，库
fs 是node.js 文件系统操作库，为我们提供读写文件的相应功能

readFileSync fs读取文件的api，一定会等到读取完文件才执行下一条语句
readFile是非阻塞的， 它会立即返回让程序继续执行。而立即这点时间对I/O操作来说远远不够，它会返回undefined
